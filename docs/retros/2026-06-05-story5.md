### درس جديد مُقترَح — L-004

**القاعدة:** في Next.js App Router، Server Component يحتاج بيانات من Supabase → استعلم مباشرةً بـ `createClient()`. لا تستدعِ API route الخاص بك عبر HTTP.

**لماذا:** self-referencing HTTP call يضيف round-trip زيادي (latency) + يحتاج `NEXT_PUBLIC_APP_URL` + يعقّد error handling. الـ cookie client يعمل مباشرةً في Server Components.

**كيف أطبّق:** اكتشفت في elegance pause لـ `leads/[id]/page.tsx`. الاستثناء الوحيد: إذا كنت في Client Component فتحتاج API route.
