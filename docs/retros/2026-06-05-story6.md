## ما يحتاج تحسيناً

### 🔴 Bug جديد: undefined→null mismatch عند حدود التكامل

**النمط:**
- qualification engine يُعيد `number | undefined` (مثل `result.seriousnessScore`)
- GhlCardInput interface تتوقع `number | null`
- TypeScript اجتاز في `npx jest` لكن فشل في `npm run build` (strict mode يمسك الفرق)

**الجذر الأعمق:** الاختبارات تعمل بـ `ts-jest` الذي قد يكون أقل صرامة من `tsc` في الـ build. الـ build هو المرجعية الحقيقية للـ TypeScript.

**الإصلاح المُطبَّق:** `?? null` guards في كل حقل nullable في استدعاء syncCardToGHL.

**مقترح L-005 ↓**
