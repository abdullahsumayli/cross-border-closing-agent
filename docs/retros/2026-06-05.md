## Lessons جديدة للإضافة لـ `.claude/lessons.md`

### L-001: External credentials أولاً
**القاعدة:** قبل `/3-eo-code` لأي AC يعتمد على خدمة خارجية (Stripe، Sentry، Unifonic)، تحقق أن الـ keys موجودة في `.env.local`. إذا لم تكن — وثّق "blocked on human action" في الخطة وتقدّم للكود فقط.
**لماذا:** Story 3 QA=7 لأن `STRIPE_PRICE_*_ID` لم تُنشأ → no sandbox test → lost 1 point.

### L-002: browser test أثناء `/3-eo-code`
**القاعدة:** لأي ملف `.tsx` جديد، شغّل `npm run dev` → افتح في browser على 375px → screenshot قبل commit. ليس قبل الشحن، بل قبل `/4-eo-review`.
**لماذا:** UX cap 8 في Stories 3-4 بسبب "browser لم يُفتَح." 5 دقائق توفّر bridge-gaps session.

### L-003: posthog/mailgun pattern
**القاعدة:** لأي خدمة analytics/notification خارجية في server-side: استخدم `fetch` مباشراً (guard على API key → fetch → .catch(()=>{})). لا dependency جديدة.
**لماذا:** Pattern تحقّق في email.ts + posthog.ts — قابل للتكرار في GHL (Story 6)، Tap (Story 7).
